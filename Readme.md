# Lab CSS

absolute lenghts เป็นการกำหนดค่าแบบตายตัวไม่เปลี่ยนแปลงแม้ว่าขนาดหน้าจอจะเปลี่ยนไป เช่น px\
relative lenghts เป็นการกำหนดค่าแบบไม่ตายตัว เมื่อขนาดของหน้าจอเปลี่ยนไป เช่น em \

Lab HTML

Tag Attribute Meaning
h class ใช้ระบุคลาสใน css
p style ใช้ระบุ inline css
div id ใช้ระบุ id สำหรับอ้างอิงถึง div ตัวนี้
img src ใช้ระบุที่อยู่ของรูปภาพ
a href ใช้ระบุปลายทางที่จะไป
ul hidden ใช้สำหรับซ่อนการแสดงผล
ol start ให้เริ่มต้นนับตั้งแต่ตัวที่เท่าไหร่
table width กำหนดความกว้างของตาราง
form action เมื่อทำการ submit form จะส่งค่าไปยัง...
button type เป็นการระบุประเภทการทำงาน ชนิดของปุ่ม

LAB 2 ความแตกต่าง

Tag ความเหมือน ความต่าง
hr กับ br เป็นการขึ้นบรรทัดใหม่ hr ขึ้นเป็นบรรทัดโล่ง
br ขึ้นเป็นบรรทัดที่มีเส้นกั้น
ul กับ ol การเป็นแสดงผลหมวดหมู่ลิสต์ ul เป็นการแสดงผลที่ไม่มีการเรียงลำดับ
ol เป็นการแสดงผลที่มีการเรียงลำดับ
select กับ checkbox กับ radio button เป็นกล่อง input ให้ user เลือก select ใช้ กับ option มีลักษณะเป็น dropdown ให้เลือก
checkbox เป็นกล่องให้ติ๊กเลือกได้หลายอัน
radio button เป็นกล่องให้ติ๊กเลือกได้เพียงอันเดียวในหมวดหมู่นั้น
div กับ span เป็นการคลอบ element อื่น ๆ div มีลักษณะเป็น block level
span มีลักษณะเป็น inline level
script กับ style เป็นส่วนที่ใช้ครอบส่วนที่ไม่ใช้ html script ใช้ครอบภาษา Javascript
style ใช้ครอบภาษา CSS

LAB 3 input type

type ใช้งานอย่างไร
button ทำให้เป็นปุ่มกดใช้งาน
checkbox ทำให้เป็น checkbox
color ทำให้สามารถ input สำหรับเลือกสี
date สามารถเลือก วัน เดือน ปี ได้
datetime-local สามารถเลือก วัน เดือน ปี เวลา ได้โดยไม่คำนึงถึง timezone
email input สำหรับกรอก email
file ทำให้สามารถกดเพื่อเลือกไฟล์ภายในเครื่องได้
hidden ซ่อนการแสดงผล
image ทำให้รูปเป็นเหมือนปุ่ม submit button
month เลือก เดือน ปี ไม่มี timezone
number กรอกได้เฉพาะตัวเลข
password สำหรับกรอก password จะมองไม่เห็นว่ากรอกอะไร
radio ทำให้เป็นปุ่ม radio button
range ควบคุมช่วง เช่น สไลด์บาร์
reset ปุ่มสำหรับการ reset ค่า
search ช่องกรอกสำหรับการค้นหาข้อมูล
submit ืทำให้เป็นปุ่ม submit button
tel ช่องสำหรับกรอกเบอร์โทร
text ช่วงสำหรับกรอกตัวหนังสือ
time ช่องสำหรับกรอกเวลา
url ช่องสำหรับกรอก url
week กรอกโดยใช้สัปดาห์
